## python-learning-tutedude
This repository contains Python learning assignments from TuteDude. Below is an overview of the programs in each assignment folder:

# Assignment 1
Introduction to basic Python operations and input/output.
program-1.py: Performs basic arithmetic operations (addition, subtraction, multiplication, and division) on two user-input numbers.
program2.py: Takes first and last names as input and displays a welcome message with the full name.

# Assignment 2
Conditional statements and loops.
program-1.py: Checks if a number is even or odd using a conditional expression.
program-2.py: Calculates the sum of numbers from 0 to 50 using a for loop.

# Assignment 3
Functions and mathematical operations.
program-1,py: Implements a recursive function to calculate the factorial of a number with error handling for negative numbers.
program-2.py: Uses the math module to perform calculations including square root, natural logarithm, and sine of a number.

# Assignment 4
File handling operations.
program-1: Reads and displays the contents of sample.txt with error handling for file not found exceptions.
program-2: A menu-driven program that allows users to write, append, and read data from files.

# Assignment 5
Data structures and dictionaries.
program-1.py: Implements a student marks lookup system using a dictionary with a loop to query student marks until the user exits.
program-2.py: Demonstrates list slicing operations including extracting the first 5 elements and reversing them.
